*Toxin - сайт поиска номеров в отеле*
---
## Перед запуском нужно выполнить 
1) `npm install`
2) `npm run build`
готовые файлы собираются в папке `docs`
`index.html` - сайт Toxin
`ui-kit` - ui-kit
